PATHOLOGY MICROSCOPY · PROJECT 06

Pathology Cell-State Change Analysis

An IDR0097 pipeline for finding which cells changed, how strongly they changed, and whether those changes form measurable phenotypic groups.

IDR0097Cell segmentationDINO featuresUMAPClusteringGitHub ↗

The project

The goal was to convert microscopy image tiles into a readable cell-level change map.

The pipeline processes paired image tiles, segments nuclei and cells, extracts morphology and representation features, computes change scores, then groups similar cells with UMAP and clustering. The output is not just a model score. It is a visual audit trail: overlays, ranked cells, cluster profiles, and before/after nuclear-cytoplasmic ratio plots.

Workflow

  • Loaded IDR0097 OME-TIFF tiles for timelapse and sequencing views.
  • Generated segmentation overlays and cell-size quality checks.
  • Extracted 1167-dimensional feature vectors per cell.
  • Computed cell-level change scores and DINO embedding distances.
  • Clustered cell states and compared sequencing profiles per cluster.

Why it matters

More than a notebook

The work turns raw microscopy into a reviewable analysis pipeline. Each stage leaves evidence: segmentation, features, rankings, cluster scores, and interpretable plots.

Product angle

The same structure can support pathology QA tools, assay-development dashboards, or research workflows where scientists need to inspect which cells changed and why.

Current state. Working research pipeline with saved outputs from three processed tiles. Next step would be packaging it into a small dashboard with tile selection, cluster filtering, and exportable review sets.
